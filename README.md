一个简单的化疗/护理计划生成器，使用Next.js和OpenAI API。

## 功能

1. 用户输入化疗/护理计划，生成未来半年内的所有相关日期
2. 用户可以保存为图片
3. 用户可以分享给其他人
4. 用户可以查看历史记录
5. 用户可以根据记录生成新的计划
